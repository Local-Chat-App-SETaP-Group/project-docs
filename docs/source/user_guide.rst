User Guide
=====
.. _installation:

Installation
------------

To use Untitled Project, install and run the .apk file provided.

.. _usage:

Usage
-----

1. Log-in to your profile, or sign up if you do not have one.
2. You will be prompted to join a local chatroom if you have not already.
3. Select your desired chatroom on the map, and press **Join**.
4. Congratulations!
